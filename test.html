Test - Statistical Drafting Assistant

Test Instructions

  1. Open the main website at http://localhost:8000
  2. Select a set from the dropdown (e.g., FDN)
  3. Wait for the pick order table to load
  4. Verify that initially "Rating" and "P1P1 Rating" columns show the same values
  5. Click the "Pick" button on any card row
  6. Verify that the card appears in your collection
  7. Verify that the "Rating" column now shows different values from "P1P1 Rating"
  8. Verify that the "Synergy" column shows the difference (Rating - P1P1 Rating)

Expected Behavior

  • ✓ Initial state: Rating = P1P1 Rating (same values)
  • ✓ After picking: Rating ≠ P1P1 Rating (different values)
  • ✓ Synergy shows positive/negative values based on rating difference
  • ✓ Collection updates when cards are picked
  • ✓ Pick order table re-sorts based on new ratings

Debug Information

Open the browser console (F12) to see detailed logging:

  • Collection vectors being sent to ONNX model
  • P1P1 ratings (empty collection)
  • Current ratings (with current collection)
  • Rating calculations and transformations
