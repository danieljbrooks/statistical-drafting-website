Filter Functionality Test

Test Instructions

This page tests the new filter functionality added to the Statistical Drafting Assistant.

Test Cases

  • Rarity filters: Uncheck "Rare" - should hide all rare cards
  • Color filters: Uncheck "Red" - should hide all red cards
  • # Cards: Change from 20 to 50 - should show only top 50 cards
  • Combined filters: Uncheck "Rare" and "Red" - should hide all rare red cards
  • Filter status: Should show "Filters active: X cards shown" when filters are applied
  • Filter reset: Check all filters back - should show all cards again

Expected Behavior

  • ✅ Filters should work independently and in combination
  • ✅ Filter status should update in real-time
  • ✅ "Find Card" and "New Draft" buttons should be at the top of the page
  • ✅ "Pick" buttons should still work and add cards to collection
  • ✅ Collection should update when cards are picked
  • ✅ # Cards dropdown should limit the number of cards shown
  • ✅ All filters should be enabled by default

Filter Options

  • Rarity: Common, Uncommon, Rare, Mythic
  • Color: W, U, B, R, G, Multicolor, Colorless
  • # Cards: 20 (default), 50, 100, All

Console Logs

Open the browser console (F12) to see filter debugging information:

  • Filter values being applied
  • Card counts after each filter step
  • Collection updates when picking cards

Notes

  • All filters start as "checked/enabled" by default
  • If no rarities or colors are selected, all are shown (fallback behavior)
  • Filters are applied in real-time as you change them
  • The collection is cleared when changing sets
